## 문자의 표현
```
네트워크가 발전되기 전 미국의 각 지역 별로 코드체계를 정해 놓고 사용했지만 네트워크가 발전하면서 서로간에 정보를 주고 받을 때 정보를 달리 해석한다는 문제가 생겼다.
따라서 혼동을 피하기 위해 ASCII 문자 인코딩 표준이 제정
7bit 인코딩으로 128문잘르 표현하며 33개의 출력 불가능한 제어 문자들과 공백을 비롯한 95개의 출력 가능한 문자들로 이루어져 있다.
인터넷이 전 세계로 발전하면서 ASCII를 만들었을 때의 문제와 같은 문제가 국가간에 정보를 주고 받을 때 발생했다.
자국의 코드체계를 타 국가가 가지고 있지 않으면 정보를 잘못 해석 할 수 밖에 없었다. 따라서 다국어 처리를 위해 유니코드를 만듬.
```

## 문자열의 분류
```
문자열
 - fixed length
 - variable length
  - length controlled
  - delimited

파이썬에서의 문자열 처리
 - char 타입 없음
 - 텍스트 데이터의 취급방법이 통일되어 있음

문자열 기호
 - ', ", ''', """
 - + 연결
 - * 반복

파이썬에서의 문자열 처리
 - 문자열은 시퀀스 자료형으로 분류되고 시쿠너스 자료형에서 사용할 수 있는 인덱싱, 슬라이씽 연산들을 사용할 수 있음
 - 문자열 클래스에서 제공되는 메서드 
  - replace(), split(), inalpha(), find()
 - 문자열은 튜플과 같이 요소값을 변경 할 수 없음

```
