# Django Form
```
 - 지금까지 HTML form, input 태그를 통해서 사용자로부터 데이터를 받았음
 - 사용자가 입력한 데이터가 우리가 원하는 데이터 형식이 맞는지에 대한 유효성 검증이 반드시 필요
 - Django Form은 이 과정에서 작업과 반복 코드를 줄여줌으로써 훨씬 쉽게 유효성 검증을 진행할 수 있음 

1. Form에 대한 Django의 역할
 - Form은 Django의 유혀성 검사 도구 중 하나로 외부의 악의적 공격 및 데이터 손상에 대한 중요한 방어 수단
 - Django는 Form과 관련한 유효성 검사를 단순화하고 자동화 할 수 있는 기능을 제공하여 개발자가 직접 작성하는 코드보다 더 안전하고 빠르게 수행 하는 코드를 작성할 수 있다.
 - 개발자가 필요한 핵심 부분만 집중할 수 있돌고 돕는 프레임워크의 특성

2. Django는 Form에 관련된 작업의 세 부분을 처리
 - 렌더링을 위한 데이터 준비 및 재구성
 - 데이터에 대한 HTML forms 생성
 - 클라이언트로부터 받은 데이터 수신 및 처리
